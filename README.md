# ⚽ Passa a Bola - Projeto Web

Um projeto web moderno para gerenciamento de jogadoras e acompanhamento de seguidores, desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- **🔐 Sistema de Autenticação**: Login seguro com múltiplos usuários
- **📊 Dashboard Interativo**: Visualização de dados com gráfico animado
- **👥 Gerenciamento de Jogadoras**: Adicionar, listar e remover jogadoras
- **🌐 Integração com APIs**: Consumo de dados externos (JSONPlaceholder)
- **💾 Persistência de Dados**: Salvamento automático no localStorage
- **⚠️ Tratamento de Erros**: Validações e mensagens de erro/sucesso
- **🧪 Testes Unitários**: Verificação automatizada das funcionalidades
- **📱 Design Responsivo**: Interface moderna e adaptável para todos os dispositivos
- **🎨 Animações Suaves**: Transições e efeitos visuais elegantes

## 🎨 Características do Design

- **Paleta de cores moderna** com gradientes elegantes
- **Tipografia Inter** para melhor legibilidade
- **Cards com sombras** e efeitos de profundidade
- **Animações CSS** suaves e profissionais
- **Modo escuro** automático baseado nas preferências do sistema
- **Gráfico interativo** com Chart.js estilizado

## 📁 Estrutura do Projeto

```
Sprint-Web-Dev/
├── index.html      # Página principal com sistema de login
```

- **Chart.js**: Gráficos interativos

## 🎯 Como Usar

### 🔐 **Login**
1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Use uma das credenciais de teste:
   - **admin** / **123456**
   - **user** / **password**
   - **teste** / **teste123**

### 📊 **Dashboard**
- Visualize o total de seguidores
- Gráfico interativo com dados atualizados
- Dados são sincronizados com API externa

### 👥 **Jogadoras**
- Adicione novas jogadoras ao time
- Visualize lista completa com posições
- Remova jogadoras clicando no botão 🗑️
- Validação automática de dados

### ⚙️ **Configurações**
- Visualize informações da conta
- Faça logout do sistema

### 🧪 **Testes**
- Abra o console do navegador (F12)
- Execute `runTests()` para rodar os testes unitários
